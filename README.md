# Steps to run this template
- Install Docker
- Install npm/yarn/pnpm
- Run ```docker-compose up```
- Change variables in config file if you wish
- Run ```yarn``` or ```pnpm install``` or ```npm install```
- Run ```yarn start:dev``` or ```npm run start:dev```
